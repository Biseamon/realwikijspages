Blueberry : Ch. 6.4 - Posting a Social Update

Created by Stelios Dritsas, last modified on Jul 01, 2016

However, if a Twitter message is entered, and then another social media icon is selected (via the top), the message will now be carried over into the new channel. Yet, if a duplicated message is then edited, the changes will not subsequently carry back over.

Note: Try not to publish identical social messages across each platform. The exception to this is when publishing a Blog with a social update, as HubSpot only provides a job-lot feature here.

For example, using an image for a Twitter update, reduces a lot of the text characters you can use, but this measure does not affect Facebook and Linkedin posts. Additionally, unlike Twitter and Facebook, LinkedIn does not use hashtags, so make sure not to include them for this channel. It’s most effective to be on the ‘Publishing’ page when posting an update, but this is not essential. To post an update, click on the blue button located towards the top-right hand side of the page called ‘Compose a message’.

Note: This button is similarly located in the same position for the Social Media pages on Monitoring, Publishing, Reports, and Settings.

A pop-up box will now appear allowing a Twitter message to be entered.

Along the top of this pop-up, there are also additional icons for Facebook, LinkedIn, and Google+. From here you can either compose a message for just one account, or publish it across all of them.

Through this interface you can also insert links, attach images, schedule a message, and attach messages to a certain campaign.

  1.  Links: Any links entered into a social update are automatically shortened upon publication, allowing HS to track overall clicks, and gage how popular messages are. Prior to publication though, links will always be displayed at their normal length. 

    Generally always try to include an actionable link within a social update. 


    As such, you can either paste-in URL’s from any site (HS site included), or to share a HS specific blog post or landing page, you can use the ‘Attach’ option to select the appropriate CMM link.

  2.  Images: If a link has been included within a social message, HubSpot will automatically pull through any relevant images from the destination page and allow them to be attached to the post.

    Alternatively, an image can be uploaded with a social message by selecting ‘Attach’ followed by the ‘Image’ button, and then choosing an image from the File Manager. For instructions, please refer to the ‘Images’ topic.

  3.  Scheduling: To schedule a message or post it immediately, open the ‘Schedule for Today’ link, and follow the on-screen instructions to set the desired time.


  4.  Campaign: A social message can also be attached to a campaign by opening the ‘Select Campaign’ drop-down menu, and choosing the relevant campaign. If you need to setup a new campaign select the ‘New’ link.
