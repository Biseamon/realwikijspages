Blueberry : Ch. 7.5 - Settings

Created by Stelios Dritsas on Jul 01, 2016

Clicking the ‘Settings’ tab towards the top of the page, allows you to enter the following information:

  1. Internal email name: Enter a name for internal reference (this will not be sent)

  2. Campaign: Select a campaign or create a new one ‘Add new’

  3. Email type: Email type is the category your email is sorted under a user’s email preferences, to which they can choose to unsubscribe from.

  4. Web version: Create a shareable web version of your email by clicking the slider to green (only if required). From here:

    1. Web page title: Internal name for organisation

    2. Web page URL: Select a domain as appropriate

    3. Meta description: Add a meta description to drive search engines to your page, but keep it under 160 characters

    4. Web page status: Choose from one of the three options available

    5. Web version information: You won’t need to change this
