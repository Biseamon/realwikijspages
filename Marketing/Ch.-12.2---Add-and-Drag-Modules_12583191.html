Blueberry : Ch. 12.2 - Add and Drag Modules

Created by Stelios Dritsas on Jul 05, 2016

From the editing tab you can add and swap the modules you want to use within your template.

  1. The quickest way to add a module is by clicking the ‘add row’ button towards the bottom of your page (adding a module directly above your cursor).

  2. From here, drag the module and reposition it as necessary, or place it over a module to segment it in different ways – experiment as needed.

  3. In addition, clicking the gear icon towards the right hand side of each module will present the following options:

    1. Edit options:

      1. Change the label name for your selected module (helps when dealing with many modules)

      2. Add default support text to your selected module, to help when entering further content for blogs, email, and webpages.

      3. Please note, edit options will vary depending on the module selected

    2. Make smart: make the module smart

    3. Lock Module: lock the module to prevent others from editing the HTML properties

    4. Edit CSS:

      1. Customize the HTML mark-up of your template for the following:

      2. Custom CSS Class (advanced)

      3. Custom Inline Styling (advanced)

      4. Custom Wrapping HTML (advanced)

    5. Swap module:

      1. Swap the module type for something else, like a form or global module for example

    6. Make module global:

      1. Make you module global to use it across other templates

      2. Please note, any changes to a global module will carry across other linked templates – proceed with caution

    7. Split horizontally: Split the selected module in two

    8. Add module below: Add and a group a module below the selected module

    9. Delete Module:

      1. Deletes the module

      2. Please note, deleting a module may also delete it from other templates where it is used – proceed with caution
