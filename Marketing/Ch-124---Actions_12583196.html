Blueberry : Ch. 12.4 - Actions

Created by Stelios Dritsas on Jul 05, 2016

There are some other important options you might to consider via the dropdown menu for ‘Actions’ (shown below), adjust as necessary:

  1. Save: Save the changes made to your draft but be aware this does not affect published pages.

  2. Move or rename template: Rename and more your template to the appropriate folder.

  3. Clone: Clone your template (great for when creating variations of existing content).

  4. Clone to file: Clone a separate HTML version of your template.

  5. Create a page from template: Create landing page directly from design manager (great for saving time).

  6. View revisions: Rollback your template to a previously published design.

  7. Pages using template: See where else this template is being used.

  8. Delete: Delete your template but be aware you cannot retrieve it.
