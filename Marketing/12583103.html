Blueberry : Ch. 8.5 – Adding a Website Navigation Tab (Advanced Menus) 

Created by Stelios Dritsas, last modified by Unknown User (mct) on Jul 07, 2016

To add a Website Navigation Tab, go to Content > Content Settings > Advanced Menus (from the left-hand side menu).

Under the ‘Edit a Menu Tree’ heading, select the ‘default’ drop down menu (shown below), and choose the relevant menu.

To add a website navigation tab follow the steps below:

To create a main page entry, scroll to the bottom of your page and click ‘Add a page’

This creates the following entry box:

Enter a menu label name for your new page tab.

Choose which website page your menu label will link to using one of the two ways below:

Click the ‘CHOOSE’ button and select the appropriate HubSpot page from the presented popup.

Or click the ‘http://’ drop down menu button, choose from one of the options below, and then paste in your appropriate URL.

External Website

Secure External Website

Relative Link

Email Link

For instructions on how to create a website page please refer to chapter 8.1.

Then use the drag feature 

 to reposition your page tab as either a main or subpage page entry.

Publish or save your changes by clicking the relevant button towards the bottom of the page.

Please note, you can also create a subpage entry underneath a page by clicking the ‘Add page’ icon 

, and then following steps 2-7.
