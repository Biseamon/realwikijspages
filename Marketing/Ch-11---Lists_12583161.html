To access Lists from the main dashboard, navigate to Contacts and click > Lists. Please note, you can read HS’s own guidance on this subject here.

The page will open upon the main dashboard for lists and display all the currently available static and smart lists used on HubSpot.

You can segment this view by clicking the drop down menus for ‘All list types’ and ‘All authors’ located towards the top right of this page, or by using the search bar to find an already know list.

Furthermore, you can sort the order of these lists by clicking on the column headers:

  1. Check box (more details below)

  2. Status

  3. Type

  4. Name

  5. Created

  6. Size

  7. Used in

  8. Author

  9. Folder

By selecting either a list’s individual check box, or by selecting the column headers check box (as shown above), the features below become available:

  1. Move list: move your list to a folder (you can create folders from the left-hand side menu)

  2. Delete list:

  3. Export list:

  4. Clone list: You can only clone smart lists

Lastly, be aware that the left-hand side menu bar is a shared menu bar, also available from the dashboards for workflows, forms, and lead scoring, and provides a quick way to navigate between these features.
