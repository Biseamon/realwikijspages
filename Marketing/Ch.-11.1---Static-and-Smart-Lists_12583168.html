Blueberry : Ch. 11.1 - Static and Smart Lists

Created by Stelios Dritsas on Jul 04, 2016

What’s the difference between a static and a smart list?

  1. Static lists: Static lists are created automatically when you import a list into HubSpot or they can be created by selecting a list criteria.  Once a static list is generated, the list members will remain the same unless you add or remove contacts manually (explanation via HubSpot).

    1. Common uses:

    2. Imported data from another database

    3. Email blasts

    4. Internal staff lists

    5. Event/trade show lists

    6. Manually adding contacts to workflows


  2. Smart lists: Smart lists offer a flexible way to organize contacts in HubSpot and automate your marketing actions. You can build these automatically refreshing lists based on contacts who meet certain criteria, and removing contacts that no longer meet that criteria (explanation via HubSpot).

    1. Common uses:

    2. Contact property segmentation

    3. Form completions

    4. Workflow completions
