Blueberry : Ch. 5.1 - Preparing a Blog Post

Created by Stelios Dritsas, last modified by Unknown User (mct) on Jul 07, 2016

It’s easier to have a blog post prepared in advance, and then copy it into HS, rather than creating it from scratch within the page editor. However, in order for a blog post to rank in Google, it must contain the following elements to generate organic traffic.

  1. Keywords: 1 primary, and 1 secondary search term (do not reuse terms).

  2. Title: <60 characters with spaces – must include the primary search term.

  3. Subheadings: Include the primary search term naturally.

  4.  Main body text: <1000 words – use both search terms multiple times.

  5. First Paragraph: Include the primary search term once naturally – twice if possible.

  6. Meta description: <21 words - include the primary search term.

  7. Image: You will need to prepare and upload a relevant image for your blog post, which includes the primary search term within the file name (Alt Tag). For instructions, please refer to chapter 2. 

  8. Call to Action (CTA): You will need to prepare a CTA for your blog post. For instructions, please refer to chapter 3.
