Blueberry : Ch. 5.5 - Blog Post Settings

Created by Stelios Dritsas, last modified by Unknown User (mct) on Jul 14, 2016

Access the main settings for your blog via the ‘Settings’ subsection:


Under ‘Post Information’, additional info is needed to categorise your blog post within HubSpot and Google. To do this, complete the following sections:


 

1) Title: If you have already entered text into the 'Title' module under 'Content', then this section should already be automatically filled. If not, please complete this process.

2) URL: The URL is automatically formed around the pre-existing page title found within the main editor. If empty, return to the main editor and insert a title.

3) Author: Select an author from the drop-down box, or create a new author using the ‘add new’ option above the entry box.

4) Topics: Your actions here will depend on which blog you're writing for - CMM or BB Consult. For CMM simply choose the topic 'Blogging', but for BB Consult, you’ll need to assign both a ‘Topic’ and a ‘Tag(s)’.

'Topics' are for article segmentation, like that for 'How Blueberry Solves' - you can find a full list of these here. To create a topic, simply click inside the 'Topics' section, type in the name of the topic, then press 'Create new topic'. Or, if your topic already exists, simply choose it from within the drop down menu.

'Tags' include subcategories like 'security', 'password handling', and 'ui design' etc. - they form a second layer of segmentation in addition to 'Topics'. However, tags are also created via the ‘Topics’ section, but to make it a ‘Tag’ rather than a 'Topic’ you’ll need to use this format: type in 'Tag: Example', or just click inside the topics box and choose from one of the available options

5) Campaign: For anything CMM related, just choose the Cloud Machine Manager option. To create a new campaign use the ‘add new’ button to the right-hand side of the entry box, and complete the presented instructions, or choose a pre-existing option.

6) Meta Description: Insert a prepared meta-description into the entry box. For instructions please refer to chapter 5.1. 

 7) Featured Image: Place an image within the main editor to complete this section. Note: if you’ve been following this guide through then this should already be completed. 

 8) Social Publishing: Create a short message for social media (under 94 characters), which will auto-publish to Twitter, Facebook, and LinkedIn, and include a backlink to the article upon publication.
