Blueberry : Ch. 8.1 - Creating a Website Page

Created by Stelios Dritsas, last modified by Unknown User (mct) on Jul 07, 2016

There are two ways to create a website page - choose whichever option is most suitable.

  1. It’s perhaps easier to clone a website page that’s similar to the design and layout you want, and then editing the content rather than creating a new page from scratch. 

    As mentioned on the previous page, you can find the cloning option for a website page via the below: 

    “To manage these pages there are some hidden options available to the left of the created date for a website entry, hover your cursor in this section to view buttons for Edit, Clone, and More” 

    To clone a page, simply click the clone button, choose a new name, and you’re ready to edit/delete the existing content and settings to your needs.

  2. Alternatively to create a new website page, click the blue button located towards the top-right hand side of the page called ‘Create a new website page’.

Once clicked, you’ll be redirected to a page which displays all the templates available to use.

Scroll through the templates to find a suitable design, or use the search-bar or drop-down ‘All folders’ menu to find a known template quicker.

Once found click you chosen template and enter a name – the more descriptive the better, then press ‘Create’ (below).

You’ll now be redirected to the Edit tab for your website page, where you can input your text with the Content Editor. 

For instructions on how to use Content Editor please refer to chapter 1, then once you’ve completed these steps, press save and proceed to the ‘Settings’ tab.
