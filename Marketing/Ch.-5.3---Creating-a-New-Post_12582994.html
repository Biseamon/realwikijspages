Blueberry : Ch. 5.3 - Creating a New Post

Created by Stelios Dritsas, last modified by Unknown User (mct) on Jul 07, 2016

To create a new blog post, open the dashboard page for the Blogging section, and click the blue button called ‘Write new blog post’, located towards the upper right-hand corner.

A pop-up will now appear called ‘How do you want to start your blog post?’ Select the option ‘Blank Post’, and you’ll be presented with the main editor for blog post creation.

However, before you begin there are some areas to take note of:

Firstly, look for the tabs called Edit, Settings, and Publish or Schedule - located towards upper-centre of the main editor page (below).

  • Edit: Is the main section to add and edit text.

  • Settings: Is the section for ‘post’ information and advanced details.

  • Publish: Publish the blog or schedule it for later.

When you’re ready to enter your text, proceed to the chapter 5.4.
