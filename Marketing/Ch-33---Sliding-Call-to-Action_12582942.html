Blueberry : Ch. 3.3 - Sliding Call-to-Action

Created by Stelios Dritsas, last modified on Jun 30, 2016
  1. Create a CTA:

    1. Create the call-to-action image you want to ‘slide-in’ at the side of your blog.

    2. Create the call-to-action using the CTA tool (under content), HubSpot recommends 400px by 178 px.

      2. Edit the Blog in HTML not Rich Text:

    1. Click the Source code icon in the blog editor.

    2. A box will appear with the blog’s source code.

      3. Add “id=last” to the paragraph or header tag:

    1. Locate the point in the blog source code where you would like the CTA to slide in.

    2. Add “id=last” to the paragraph or header tag e.g. <p “id=last”> or <h2 “id=last”>.

    3. For example:

      4. Staying in HTML mode (Source Code box), scroll right to the bottom of the text.

      5. Open a notepad tool and copy and paste the following code into it:

    1. <div id="slidebox"><a class="close">&nbsp;</a>ADD CTA EMBED CODE HERE</div>.

      6. Go to the call-to-action tool in HubSpot > Click on the Cog next to ‘Details’ and click embed.

    1. Copy the Embed code and paste into your open notepad tool.

    2. Replace the ‘ADD CTA EMBED CODE HERE’ text with the pasted embed code from HubSpot.

    3. Copy the resulting code.

      7. In your blog’s source code, paste the code at the very bottom.

      8. Save and update the page and check if it works.
