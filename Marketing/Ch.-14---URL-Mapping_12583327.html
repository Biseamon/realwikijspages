To access URL Mapping navigate to Content and click > Content Settings. Please note, you can read HS’s own guidance on this subject here.


From here look to the left hand side for a link called ‘COS URL Mappings’ > click it. From here you can edit or create new URL mappings.

To create a new URL mapping:

To create a new URL mapping press, the ‘Add a new redirect’ button towards the top right hand side.

From here, simply add the URL address which you wish to redirect to ‘Original URL or path’ and add the new destination URL to ‘Destination URL or path’. In most cases, you can leave the ‘Redirect Type’ to ‘Permanent (301)’ - I’ve never had a need to change it.

To edit an existing URL mapping:

The easiest way to find and edit an existing URL (rather than searching page by page), is to paste the exact web address into the search bar and hit go. Once found, you can edit the URL by clicking the ‘gear’ icon underneath the ‘Actions’ column and pressing ‘edit’. 

From here you can change the ‘Original URL or path’ or ‘Destination URL or path’ as needed - but please make sure any changes made are accurate. In most cases, you can leave the ‘Redirect Type’ to ‘Permanent (301)’ - I’ve never had a need to change it.
