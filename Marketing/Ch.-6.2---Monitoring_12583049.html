Blueberry : Ch. 6.2 - Monitoring

Created by Stelios Dritsas on Jul 01, 2016

The Monitoring page displays what Twitter communications there has been for specific search phrases called ‘Streams’.

Streams are user created and effectively cast a net across Twitter, pulling-in any tweets that contain a particular set of words. Highly useful for monitoring trends and competitors!

You can view an already created Stream either by selecting a named tab like those shown below, or by clicking the ‘More’ option towards the far right-hand side, which shows every stream available.

By selecting a stream, it will display what discussions there have been for its terms, allowing you to monitor areas of interest, brand chatter, and competitor activity.

To create a new stream, click the ‘+’ icon next to the previously discussed ‘More’ link, and follow the on-screen instructions. To edit a current stream, select that particular stream and click ‘edit’ (below), then adjust or add new search phases to the ‘that contain’ text box, then save.
