Blueberry : Ch. 3.1 - Creating a Call-to-Action

Created by Stelios Dritsas, last modified by Unknown User (mct) on Jul 07, 2016

To create a new CTA, click the blue button located towards the top-right hand side of the page called ‘Create a CTA’.

On the presented pop-up, fill out the following details under the ‘Information’ tab.

For reference, you’ll also see a preview of how your CTA looks during development further down in the pop-up, so keep an eye on this as you complete the following.

  • Name: Give your CTA an internal name.

  • Campaign: Attach your CTA to a campaign, or create a new campaign with ‘Add new’.

  • Link to: Choose to link the CTA to a web address or an existing HS page.

    • Link URL: Place a web address here.

    • Redirect to Page: Choose an existing HS page here.

    • Open in new window: Always tick this.

Fill out the following information under the Design tab.

  • Choose the type of CTA button you want, Custom or Image.

 

For a Custom Button:

  • Style - choose the CTA’s style.

  • Background Colour: Choose the CTA’s colour.

  • Size: Have the CTA sized automatically with padding options (recommended), or manually choose the width and height.

  • Text: Choose the CTA text.

  • Create CTA: Finish the CTA by clicking here.

For an Image Button:

  • Image: Select the Choose Image link, and use the File Manager to select an image (to upload an image please refer to chapter 2).

  • Image Size: Adjust the image size accordingly.

  • Alt Text: Add your alt text.

    • Note: For a blog post CTA, include the primary search term here.

  • Create CTA: Finish the CTA by clicking here.
