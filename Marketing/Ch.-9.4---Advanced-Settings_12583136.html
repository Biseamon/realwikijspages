Blueberry : Ch. 9.4 - Advanced Settings

Created by Stelios Dritsas on Jul 04, 2016
  1. Page settings

    1. Edit Head HTML: Add raw HTML snippets, embed codes, or tracking java scripts that will be added within the tag on your page.

    2. Edit Foot HTML: Add raw HTML snippets, embed codes, or tracking java scripts that will be added within the tag on your page.

    3. Password protect: Password protect your page so only specific customers can view it

    4. Set page expiration: Set an expiration day and time, with a redirection destination for viewers after expiry  

  2. Template settings: Edit or swap you landing page template, but please note any changes you make might also affect other pages, and therefore this is not a recommended action. However if still needed, clone and save a template under a new name, then swap your landing page template for your cloned design and edit as necessary.

  3. Style settings: Due to my limited develop knowledge, I cannot explain the style sheet settings at this time

Once you’ve completed these steps, press Save and proceed to the ‘Publish or Schedule’ tab.
