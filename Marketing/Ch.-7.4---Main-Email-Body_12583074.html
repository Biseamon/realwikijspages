Blueberry : Ch. 7.4 - Main Email Body

Created by Stelios Dritsas, last modified by Unknown User (mct) on Jul 07, 2016

Scrolling further down the page you’ll see your email, and all the main editable sections if you hover over them. You should only really need to change the ‘body’ modules which allow you to enter and edit your text, and ideally leave the header and/or footer image alone (which you should’ve completed at the template stage).

HTML

To edit an HTML version, select the HTML tab (as shown above).

To edit a text module, simply click inside the greyed out boxes with the pencil icon, and enter your text. However, if you already have your text prepared within a Word document, make sure to avoid any formatting issues by cleansing it within Windows notepad before implementing it within your email.

Some suggested things you may want to implement within your email are:

  1. Images;

  2. Links:

  3. CTA’s

  4. Personalisation tokens

  5. Source code

For help on how to implement these areas, please refer Content Editor in chapter 1. Then once you’ve completed these steps, press save and proceed to the ‘Settings’ tab.

Plain Text

To edit a Plain text version, select the Plain text tab (as shown above).

Please note, a plain text copy of your email is always created alongside your HTML version. To edit this copy, click the Plain Text button (show above) above the text box to the top-right, then click edit to make your changes, or revert the text back to its HTML copy by selecting the Reset from HTML button from the same position as the Edit button (below).

 

Once you’ve completed these steps press save and proceed to the ‘Settings’ tab.
