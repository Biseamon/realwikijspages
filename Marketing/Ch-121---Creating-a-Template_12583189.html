Blueberry : Ch. 12.1 - Creating a Template

Created by Stelios Dritsas, last modified on Jul 05, 2016

To create a new template, look towards the upper right-hand side of your page and click the blue button called ‘New template’.

From the opened popup select the ‘Template Builder’ option, then choose your content type from the options below, then create.

  1. Page (for website and landing pages)

  2. Email

  3. Blog

  4. System (ignore)

Please note, there are some differences in what modules you can utilise between each template type, but the process is mostly the same.

Once you’ve clicked ‘Create’ another popup will open for ‘Save new page template’ where you can enter a ‘File name’ and choose which folder to sort your template into. If there isn’t a folder to sort your template into, create one by clicking the ‘New folder’ button.

When finished press ‘Save’ to be forwarded to the editing tab.
