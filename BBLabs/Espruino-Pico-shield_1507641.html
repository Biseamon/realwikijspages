Steps to place a PCB order





Follow the steps below to place an order for the Pico or Photon shield.

Software used to create the PCBs: DesignSpark  PCB 7.0

 

1. Go to PCB train 

2. Log in using the details stored in LastPass (Shared-MISC folder)

3.  Go to PCB Fabrication



 

 

4. Choose the option highlighted below



 

 

5. For the circuit size enter:  Circuit Size X (mm) = 21 and Circuit Size Y (mm) = 36 (or 37 if ordering the Particle shield)



 

 

6. Choose the quantity. We usually choose x5 and 15 day delivery as it's the best value for money.



 

7. Enter a PCB Ref



 

8. Add To Basket



 

9. Dismiss the next prompt



 

10. Go to the checkout



 

11. Select a billing address (use this as the delivery address as well)

 



 

12. Select the delivery service

 



 

13. Choose the zip with corresponding to the PCB you wish to order (Pico or Photon) and click Upload.

Note: you can download the Espruino Pico PCB here and download the Particle Photon PCB here.



 

14. Follow the next step to review the order details and proceed to payment.

DONE!
