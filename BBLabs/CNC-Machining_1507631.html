Here is a tutorial on how to operate the Heiz S-400 CNC machine.

 

Before running a job


1. Make sure the machine is clean

  • Wipe the rails and lead screws

  • Lubricate them if necessary

  • Check the zero sensor is plugged in correctly


2. Switch ON the machine

3. Open WinPC-NC

4. Press F8 - the machine will move to set the zero coordinates

Running a job


1. Transfer the .NC file on the CNC PC

2. Open WinPC-NC and load the .NC file

3. Move the Z axis to your desired start point

4. Click the Start button (next to Move)
