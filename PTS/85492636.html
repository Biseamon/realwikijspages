Blueberry Group : How do I use Excel to Import and Export Tasks? 

Created by Jason Cozza on Aug 24, 2020

PTS allows the tasks for a project to be imported for and exported to .xls files.

XLS File Creation

Download the task xls template file from  here.

  1. Enter the project name and code where indicated by 'xxxxx' and enter the project shortcode as the worksheet name (also indicated by 'xxxxx')
  2. Leave the Task ID column blank - this will allow the new tasks to be allocated a task number during the upload process
  3. Use the Level 1 - 5 columns to create the tree structure using the names of each task
  4. Enter any additional information into the other columns provided. Please note - where a reference to an employee is required (Responsible, Team, etc), enter their Short Name

     


 


 

Uploading the XLS File

To upload the tasks in PTS:

  1. Login to PTS
  2. Go to the All Tasks page for the appropriate project
  3. Click the 'Import' button
  4. Select the xls file
  5. Click the 'Send Stats Report' checkbox - this will notify you when the upload is complete and of any errors that have occurred during upload.
  6. Click OK

Alternatively, to upload the tasks by email attach the xls to an email (note - only one attachment per email is supported) and send the email to import-task@pts.bbconsult.co.uk or import-issue@pts.bbconsult.co.uk. The email should not contain anything apart from valid PTS commands.
