Blueberry Group : PTS Outlook Client

Created by Jason Cozza on Aug 24, 2020

BB Outlook Tools includes a feature that allows PTS integration with Outlook. BB Outlook Tools can be downloaded here.

PTS Client Integration

Issues and Tasks in PTS can easily be updated using emails but this requires remembering all of the necessary tags that could be needed. PTS Client Integration makes this even easier by providing an easy interface with the most commonly used options on it.



To use this feature, click File > BB Outlook Tool, click Feature Tools and then enable the 'PTS Client' checkbox in the General Options tab.

You can use the Status and Priority drop down menus to update these fields and the Add and Delete button to update the Team members. Pressing Commit will then open a reply to the PTS email with the tags added and the bottom. Simply add any comments as necessary and send the reply to update the Issue/Task.

If you want to create a new Issue / Task instead of updating the current one, use the New PTS Mail and its associated options to do this - selecting a type of entity and a project will allow the correct email address to be auto-generated.

The Debug option will send log files for BBOL to Kevin Lazenby.

Installing PTS Outlook Client

To install PTS Outlook Client, simply download the latest version of BB Outlook tools here and then run the installer found in the zip file.
