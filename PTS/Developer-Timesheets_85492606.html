Developer Timesheets 

Setting Working Time

Your working time will allow managers and other developers to see when your are available. To set your working hours:

  1. Open the 'My Details' menu.
  2. Click 'My Timesheets'.
  3. Find the day that you are interested in.
  4. Highlight the hours that you are available for on the timesheet.
  5. Right click and click the ‘Mark selected as working time‘ button next to Planned Time.

If you will be available for the same hours each day in a week, simply:

  1. Allocate the working hours for a single day.
  2. Click the ‘Copy selected day‘ button.
  3. Select the next day and click ‘Paste selected day‘.
  4. Repeat for each day in that week

When you have set your working time for each day in a week, you can then use it to create a template for your working hours that will automatically appear on your timesheet for each week. To do this:

  1. Complete the working hours for a single week.
  2. Click the 'Set current week as a template' button.

Booking Time for Work Done

To book time to work done on either a Task or an Issue, do the following:

  1. Open the 'My Details' menu
  2. Click 'My Timesheets'.
  3. Click the 'Entity' button next to Timesheet Items.
  4. Select 'Entity' from them 'Type ' drop down menu (this should be selected by default).
  5. Enter the Task / Issue number in the 'Work Item' field.
  6. Use the Start and End fields to enter the time that you spent working on that item.
  7. Click Save.

When the item is saved, you will see a block of time appear on the timesheet UI that represents the time spent working on that item.

Alternatively, the time spent can be added to the timesheet by dragging it from the list of tasks and issues that appear on the right side of the screen. The time booked will begin where the top left corner of the dragged item is placed. This can be changed by simply dragging the block of time around the timesheet. Similarly, the period of time that the work took can be changed by stretching the block of time. Double clicking on the block will open a popup window that will also allow you to edit the start and end time of the work done.

A final method to set the time for the work done is to highlight the correct time on the timesheet and then drag the task or issue onto the highlighted time - the block of time will automatically be set to the highlighted time.

 

Other Timesheet Items

If you need to book time to something that is not either a task or an issue, the following items are also available:

  • Paid Holiday - Permament employees can book paid holidays from their annual allocation of holidays days. For more details on holidays, go here.
  • Unwell - Use this if you have been unable to work because of illness. For more details on sick leave, go here.
  • Other Discussion - Use this for work that is not part of a project to which you have been assigned. We should aim to track as much work against a project as possible.
  • Unpaid Leave - This should be used rarely and with agreement from your project manager. This is for marking time that you are planning not to work but aren't expecting to be treated as Paid Holiday.
  • No Work Assigned - This is expected to be used very rarely. This is used to indicate that you were not given work by your project manager. If you have less than 3 days work lined up ahead of you it is your responsibility to tell your manager that you are running out of work. Remind them each day until they find you some work. If you actually run out of work then you should mail the two managing directors Martin Green (martin.green@bbconsult.co.uk) and David Lee (david.lee@blueberrysystems.co.uk). If you don’t have work at the end of that day, then something has probably gone wrong with email. In that case, use Skype, phone, and any other method you can think of to get hold of a manager. As long as you follow that process then we’ll have no objections to paying you for time you spent waiting for new work to be assigned. We do expect you to try and make good use of that time, e.g. by studying existing code.

All of these items are added by selecting the correct day and then clicking the relevant button at the top of the timesheet.

Alternatively, select the time on the timesheet, click the Entity button next to Timesheet Items and then select the type of item in the Type drop down menu.

 

State Holidays

If a day is a state holiday that you are eligible for, the day will be highlighted in red and a set of additional buttons will appear at the top of the timesheet.

  • Take selected holiday - you intend to take the holiday and not work
  • Reserve selected holiday- you intend to work on this day but will take another day as holiday instead
  • Take holiday as a work day - you intend to work on this day and want to be paid for it

If you choose to reserve a holiday an additional option will become available:

  • Take reserved holiday - you intend to take a holiday on this day instead of the usual state holiday

     

Creating an Invoice

Important: Invoices should be created and sent for confirmation every calendar month. Invoices should be created at the end of last working day of month or on the first day of next month. Please note: You should invoice every calendar month, even if you've just started with Blueberry and worked only a few hours in the month.

To submit a invoice for payment, you first need to have items booked to your timesheet. You can then submit an invoice from the Invoices screen (http://pts.bbconsult.co.uk/EditBBEmployee.aspx?page=Invoices) by doing the following:

  1. Open the 'My Details'.
  2. Click 'Personal Details'.
  3. Click 'Invoices' in the menu on the right side of the screen
  4. Click on the Create Invoice button - this will add a new invoice entry in the grid.
     
  5. Click on the Edit link for the new Invoice.
  6. Select the correct Month that you wish to be paid for and then click Refresh Data. Note: it is necessary to select Month only for your first invoice. For rest of invoices you will need to Refresh Data only.
  7. Add any Adjustments or Allowances as necessary (for example, equipment allowance or monthly internet allowance).
  8. Click the Send Invoice button.

  or

  Click Close Details to return to the main Invoices screen and click the Send link next to the relevant Invoice (Note: Until the Send link is clicked, it is still possible to edit the invoice)

Please note, that page with invoices is also accessible via menu: My Details / My Invoices.

It is possible to delay payment for invoices. This feature can be used if you would like to get payments for several invoice at once. You can do this by setting 'Delay Payment' flag for invoice. You will get invoices paid, when you clear this flag.
