Blueberry Group : Can I book time to an Enquiry? 

Created by Jason Cozza on Aug 24, 2020

  1. Open PTS
     
  2. Go to the 'Edit Enquiry' page for the enquiry in question
  3. Find the 'Is shown in Shortlist' option at the bottom of the page and enable it
     
  1. Save changes

Then you need to add it to your Project list: 

  1. Open PTS Client
     
  2. Go to the My Projects -> Project Shortlist
  3. Click the Add Project button in the bottom corner

Search for the enquiry and then click the + button next to it

All enquires that have been enabled to be shown in the Project Shortlist will now appear and can be added to your own personal project list allowing time to be booked to them exactly like a project.
