Blueberry Group : What reports are available from PTS? 

Created by Jason Cozza on Aug 24, 2020

PTS contains a number of different reports. These are detailed below.

Project Task Progress

Project Task Progress
This report allows you to view the tasks (and optionally the Issues too) for a Project. The time spent on each task can be compared to the estimated time. Tasks are sorted by parent tasks, child tasks and issues (moving from left to right across the first 3 columns). Tasks can also be grouped by Scheduled. An overall summary is given at the bottom of the report.
 

Task Reports

Estimates Report
Creates an Excel report in one of 3 formats – ‘Show Issues in Detail with Total Row’, ‘Show Issues Summarized’ and ‘Show Just Task Totals – No Issues’. Each report includes the time spent on each task or item and the estimated time, as well as the Status of each task. Tasks can be filtered by Status if desired.

 



Tasks Report
Creates an Excel sheet that details all tasks for a project. A very simple report that just displays the contents of all selected fields (customisable when running the report) for each task.

 

Timesheet Reports

Timesheets
Displays the timesheets for all employees over a 4 weeks period.

 



By clicking on a date, a more detailed view of that week can be seen in one of 3 views – simple view (See below), calendar view or detailed view.
 


Timesheets By Projects
Shows the hours spent by employees who have booked time against any project against all of the projects that have time booked to them for any set period of time along with their total time spent during that period.

 


Weekly Timesheets By Project
Similar to the Timesheets report but breaks down the hours spent into weeks on a per project basis.
 


 

Revenue Report

Revenue Report
Displays revenue by month for each project over a set period of time, as well as the total revenue for that same period. Columns are colour coded to indicate past months, current months and future months. Can be run for Projects or Enquiries.
