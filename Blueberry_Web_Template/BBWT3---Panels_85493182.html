Blueberry Group : BBWT3 - Panels

Created by Jason Cozza on Sep 21, 2020

Overview

This section deals with how to implement panels within BBWT3.


Accordion element consists of one or more p-accordionTab elements. Title of the tab is defined using header attribute.

Example:

Example HTML:

<p-accordion>
    <p-accordionTab header="Godfather I">
        Content 1
    </p-accordionTab>
    <p-accordionTab header="Godfather II">
        Content 2
    </p-accordionTab>
    <p-accordionTab header="Godfather III">
        Content 3
    </p-accordionTab>
</p-accordion>


Panel is defined with p-panel element.

Example:

Example HTML:

<p-panel header="Godfather I">
    Content
</p-panel>

For panel element outside of an accordion, is recommended to use it in forms where we’d like to logically group controls without hiding any.
