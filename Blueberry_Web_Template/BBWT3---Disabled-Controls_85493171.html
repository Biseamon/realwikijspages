Blueberry Group : BBWT3 - Disabled Controls

Created by Jason Cozza on Sep 21, 2020

Overview

This section covers how to disable the interactivity of a control, in this case a text box.


To disable a control use the property "[disabled]" and set it to "true".

Example HTML:

<div class="ui-g">
    <div class="fixed-label">
        <label for="textField">Label</label>
    </div>
 
    <div class="ui-g-12 ui-md-4">
        <span class="md-inputfield">
            <input pInputText id="textField" [disabled]="true" />
            <label>Edit Text</label>
        </span>
    </div>
</div>


More disabled control examples can be found here.
