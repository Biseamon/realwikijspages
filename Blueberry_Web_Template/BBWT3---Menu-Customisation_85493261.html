Blueberry Group : BBWT3 - Menu Customisation

Created by Jason Cozza on Sep 21, 2020

Overview

BBWT3 allows you to easily change the order of pages and their position on the site.

  • Create new pages.
  • Re-arrange existing pages.
  • Delete pages.
  • Customise names of pages.
  • Edit footer pages.
  • Edit footer order.

Test Case

  1. Admin receives feedback from users.
  2. Users are finding it difficult to navigate to a certain page easily.
  3. Admin opens up "Menu Configuration" and is able to rearrange items to make them more easily identifiable for users.

For Testers

  1. Go to the BBWT3 menu bar and click on "Operational Admin".
  2. From the drop down menu click on "Menu Configuration".
  3. Drag and drop pages (including in the "Footer Menu").
  4. Changes take effect immediately.

For Project Managers

  • Quickly create, manage and edit pages.
  • Add new footer pages, and re-arrange them with dragging and dropping.

For Marketing

  • Easily create, edit, and delete pages.
  • Re-arrange pages easily through dragging and dropping.
