Blueberry Group : BBWT3 - Server Info Dialog

Created by Jason Cozza on Sep 21, 2020

Overview

Generates static info of server data such as;

  • Server name.
  • Full build number.
  • Server and Client IP's.
  • Operating system.

Test Case

  1. Admin is receiving reports of broken functionality from users.
  2. Admin tracks down that the issues are coming from one particular server.
  3. Admin can quickly check which build version is running to see if it is of an old build.

For Testers

  1. Go to the bottom footer of the page and click on the (I) icon.

For Project Managers

  • Quickly see server information such as IP and name of server.
  • Able to see full build number.
  • Able to see full server OS build directly from browser.

For Marketing

  • BBWT3 displays valuable server information such as; IP address, operating system, server name, version number and server name.
  • All this can be a valuable asset when attempting to track down a problem.
  • For instance you may have multiple servers running the web application, and one of them is not performing correctly. BBWT3 can display the aforementioned data which may lead to resolving the issue.
