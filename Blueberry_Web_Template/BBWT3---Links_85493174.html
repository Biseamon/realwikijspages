Blueberry Group : BBWT3 - Links

Created by Jason Cozza on Sep 21, 2020

Overview

You may want to have links in a form, this page describes with examples how to code for them.


 A link

Example HTML:

<div class="ui-g">
    <div class="fixed-label">
        <label>Label</label>
    </div>
    <div class="ui-g-12 ui-md-4">
        <a href="app/demo/guidelines/links">A link</a>
    </div>
</div>


Anchor links (href="#foo") must include the full path even when linking within the page. Example:

In page anchor link
Radiobutton section on labels and inputs page

Example HTML:

<a [routerLink]="['/app/demo/guidelines/links']" (click)="onAnchorClick()">Links section</a>
<br/>
<a [routerLink]="['/app/demo/guidelines/inputs']" (click)="onAnchorClick()">Radiobutton section on labels and inputs page</a>
