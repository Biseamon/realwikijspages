Blueberry Group : BBWT3 - Photo/Image Management 

Created by Jason Cozza on Sep 21, 2020

Overview

Many applications need to store photos. BBWT3 includes a photo management module which stores files securely within Amazon S3 (it can be switched to Azure Object Storage in future).  It supports upload of new files, display in the application GUI of file details, and download.

  • Ability to upload files.

Resizes file to save server space.

Ability to rotate and flip images prior to uploading.

Ability to view uploaded pictures.

  • Ability to download existing uploaded pictures.

Ability to delete existing uploaded pictures.

Test Case

  1. User is required to upload a profile image.
  2. User can upload to their profile their picture and make minor adjustments.

For Testers

  1. From the BBWT3 context menu click on "Feature Demo Pages".
  2. Click on "Photo Management".
  3. Click on "Choose" and locate the file you wish to upload.
  4. If adjustments need to be made you can click on rotate or mirror as needed.
  5. Click "Upload".
