Blueberry Group : A10:2017 - Insufficient Logging & Monitoring

Created by Jason Cozza on Sep 22, 2020

Overview

According to OWASP, insufficient logging and monitoring, coupled with missing or ineffective integration with incident response, allows attackers to further attack systems, maintain persistence, pivot to more systems, and tamper, extract, or destroy data. Most breach studies show time to detect a breach is over 200 days, typically detected by external parties rather than internal processes or monitoring.

Example

An attacker runs several scans of the web server hosting the app to gain useful information.

An attacker uses this gained intel to find a vulnerability within the web application.

They managed to gain admin access through the vulnerability and implant malware that gets viewed by the web application's users.

This malware goes undetected so more users are effected by the malware without realising they are becoming apart of a botnet.

The attacker is able to build up a sizeable amount of infected computers before the sysadmins realise that there is malware on their server.

How BBWT3 Mitigates This Vulnerability

  • Extensive logging through GrayLog.
  • Automatic alerts.
  • ReCaptcha.

BBWT3 uses a combination of GrayLog as well as Raygun services to monitor and log behaviour on the web application. Using these powerful tools we are able to discern what has happened or is happening as well as being alerted to potential problems. Not only do we have these 3rd party tools, but there are admin facing features which display logs as well. Logs for changes made to the web application as well as login logs. Giving you the information at a glance of what has happened, and giving us detailed information that we can dig through to resolve any issues. Detected errors are recorded and sent back to the BBWT3 project team as a ticket, which immediately notifies the development team of the issue.

ReCaptcha can be configured which will enable us to detect suspicious activity at login, which can alert admins to a potential attack.
