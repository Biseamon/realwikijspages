Blueberry Group : BBWT3 - Password Managers

Created by Jason Cozza on Sep 21, 2020

Overview

BBWT3 supports a variety of the most common password mangers including LastPass and 1Password. Password managers are a valuable way of storing strong user passwords in one place. The passwords stored within the manager have to be accessed by a master password - this enables a user to only have to remember 1 strong password. In turn this keeps all passwords safe as site specific passwords can all be different and be made of an assortment of characters (using a password generator), but the user only needs to remember the 1 master password.

Test Case

  1. User registers and account with BBWT3.
  2. The accounts password can be a random assortment of characters.
  3. The user does not need to remember the password for BBWT3 as it is stored in a password manager.
  4. After registration is complete user can login using the data stored within their password manager.

For Testers

  1. Go to BBWT3 login page.
  2. Click on "then sign up".
  3. Fill out details.
  4. Click on "Add".
  5. A message should be displayed stating that the password has been saved by the password manager, or a prompt to do so.
  6. Go to login page.
  7. You should be able to click on an icon near the username/password field to fill it with the details held by the password manager, if the fields are not automatically populated.

For Project Managers

  • Users can keep their account credentials stored within a password manager.
  • Using a password manager helps users to use strong passwords that they don't have to remember themselves - they only have to remember one strong master password.
  • This keeps accounts on BBWT3 more secure, by deterring the user from using passwords that are used across multiple sites.

For Marketing

  • BBWT3 has password manager support, including some of the biggest names such as LastPass and 1Password.
