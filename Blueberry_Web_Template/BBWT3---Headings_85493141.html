Blueberry Group : BBWT3 - Headings

Created by Jason Cozza on Sep 21, 2020

Overview

Below are some examples of what each heading should look like.


The heading above is an <h3> tag:

<h3>Headings</h3>

Here is an <h3> tag, which we use at the top of the page.

<h1>Page Title</h1>


In some sites we want a sub-part of the heading, which indicates an object ID or similar. We do that with a <small> inside the <h1> tag:

<h1>Edit <small>Brake Part #23445</small><h1>

Prefer tabs to header sections when the page will see lots of use by inexperienced users.
