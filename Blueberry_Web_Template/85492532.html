Blueberry has invested time in developing the Blueberry Web Template (BBWT). This is a collection of pre-built modules, programming practices and design styles which make it easier to create powerful and easy to use web applications using Microsoft ASP.NET, C# and jQuery.

A Getting Started on BBWT Development guide can be found here.

We have an online demo here - http://bbwt.demo.bbconsult.co.uk – please contact us for a user/password to access this site.

For a more detailed overview of the features it supports see the documents in https://svn.bbconsult.co.uk/repository/bbwt/docs .

BBWT has a built in mechanism to allow the testers to give feedback. If a user is working with a form or page and notices something they would like to question or comment on within the page itself, they can do this by clicking on the pushpin icon:

..in the bottom RHS of the page. This brings up a form that the user can fill in with their comments. They can then drag the pushpin icon to point to the item in question on the page. The example below show this:

Figure 9  Showing the user making use of the feedback feature to highlight an issue on a form that they would like to question. When the user has
