Blueberry Group : BBWT3 - Email Templates

Created by Jason Cozza on Sep 21, 2020

Overview

Ability to set up and edit existing email templates. You can send a test email via the cog wheel, edit an existing template email or delete an existing template. The template definitions are found in "Email Template Parameters" section when editing or making a new template email.

Test Case

  1. Admin wishes to set up company branded email's.
  2. Admin has ability to quickly change formatting and add branding.
  3. All emails now are sent with company branding in them.

For Testers

  1. Click on "Operational Admin" from the BBWT3 menu.
  2. Click on "Email Templates".
  3. To; add a new email click on "Add", to alter an existing email click on the pencil icon, to remove a template click on the bin icon.

For Project Managers

  • With modern email, we need to use a password-protected account to send email – BBWT3 settings include this option. 
  • BBWT3 includes an Email Template editor which allows the administrator to edit the text and formatting used when sending an email.  Allowing for example, some branding to be introduced.
  • When constructing a new email template it can be done quickly and effectively by using template parameters.

For Marketing

  • With templates already pre built the site is up and running as soon as possible.
  • Ability to customise templates, thanks to simple template parameters, templates can be changed quickly and new ones made easily.
  • Easily customise your templates with your own branding in a professional manner.
