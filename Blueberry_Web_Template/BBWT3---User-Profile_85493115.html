Blueberry Group : BBWT3 - User Profile

Created by Jason Cozza on Sep 21, 2020

Overview

The User Profile feature contains information about the user e.gÂ  email address, photo and name. From here users can upload a picture of themselves, correct any errors about their information as well as set up 2 step authentication.

  • Contains basic user information such as name.
  • Contains contact details.
  • Has picture upload functionality.
  • Allows access in setting up 2FA.

Test Case

  1. User's identifies that information is incorrect about them.
  2. User does not need to notify admin to make changes to their account.
  3. User goes to their profile and is able to make corrections.

For Testers

  1. If on mobile or tablet first click on the top right 3 horizontal bars.
  2. Click on the top right user image.
  3. Click on "Profile" from the drop down menu.
  4. Make whatever changes need to be done.
  5. Click "Save" to save changes.

For Project Managers

  • Users are able to correct any information by themselves on their profile as needed, preventing the need for email correspondence with an admin.
  • This saves time and is trivial to perform so users should have no issue in resolving mistakes as soon as they are identified.
  • Users are able to upload profile pictures.
  • From the profile page users are also able to set up 2 step login authentication, which is encouraged to increase security.

For Marketing

  • BBWT3 allows its users to correct information on their profile by themselves - saving time when compared to needing to email an admin to make minor changes to details.
