Blueberry Group : BBWT3 - File Upload

Created by Jason Cozza on Sep 21, 2020

Overview

File upload system allows users to upload files with additional support of drag/drop from file system.

Test Case

  1. Client requires users to be able to upload images for certain events - such as receipts.
  2. User can take a picture on their phone of receipts.
  3. User then is able to navigate to "Photo Management" and upload picture.
  4. User can then make changes as needed such as orientation.

For Testers

  1. From the BBWT3 context menu click on "Feature Demo Pages".
  2. Click on "Photo Management".
  3. Choose the location of a file you wish to upload.
  4. Select whether the image needs to be rotated/flipped or not.
  5. Click "Upload".

For Project Managers

  • Inbuilt file uploader for uploading files to server.
  • Drag and drop functionality.
  • Limitation: Drag and drop does not support multiple files.

For Marketing

  • Upload photos without the need for 3rd party tools to make minor adjustments like rotation or mirroring.
  • Automatically resizes uploaded image to save on server space and bandwidth.
