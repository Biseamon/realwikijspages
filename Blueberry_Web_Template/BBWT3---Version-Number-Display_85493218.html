Blueberry Group : BBWT3 - Version Number Display

Created by Jason Cozza on Sep 21, 2020

Overview

Displays the build number in the bottom right corner of the footer.

  • Easily visually identifiable.
  • Full build can be seen by accessingÂ Server Info Dialog.

Test Case

  1. User is in a development environment and discovers an issue with BBWT3.
  2. User can check which version of BBWT3 is being used and report the issue.
  3. Developer can then check whether the issue is due to an older build which has now be addressed, or if it is a new issue in a current build.

For Project Managers

  • Useful for keeping track of build's and bugs.
  • It is automatically generated by GitLab.
  • Can also be tracked in PTS.

For Marketing

  • Allows for testers to track changes that have occurred and ensures that there is no confusion; for example, testing on an older build than the current one could lead to wasted time and in turn money.
  • Having the version number there allows for everyone to known if it is the latest build available.
