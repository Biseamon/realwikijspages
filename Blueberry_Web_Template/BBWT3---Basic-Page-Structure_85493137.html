Blueberry Group : BBWT3 - Basic Page Structure

Created by Jason Cozza on Sep 21, 2020

Overview

This page outlines the agreed structure that all pages must follow.


All pages should have a title (displayed on the page) that matches the menu item used to select it.

All pages should have a simple description below the title.


Here is the basic layout of a page:


Example HTML:

<div class="ui-g">
    <div class="ui-g-12">
        <div class="card card-w-title">
            <h1>Page Title</h1>
            <div class="ui-grid ui-grid-responsive ui-fluid">
                <div class="ui-g-12">
                    <label for="name" class="ui-grid-col-2 control-label">Label</label>
                    <div class="label-content ui-grid-col-3">
                        <span class="md-inputfield">
                            <input pInputText id="textField" class="form-control"/>
                            <label>Edit Text</label>
                        </span>
                    </div>
                </div>
                <div class="ui-g-12">
                    ...
                </div>
            </div>
 
            <div class="ui-grid-row">
                <button pButton type="button" label="Save"></button>
                <button pButton type="button" label="Cancel"></button>
            </div>
        </div>
    </div>
</div>
