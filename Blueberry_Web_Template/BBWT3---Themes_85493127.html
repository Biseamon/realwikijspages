Blueberry Group : BBWT3 - Themes

Created by Jason Cozza on Sep 21, 2020

ThemesÂ are the colour themes available in an application. This is different to a template, which provides the structure of the client application HTML.

Overview

Themes can be selected based on the users' preference. Ultima currently provides twelve themes to chose from.

  • Ability to quickly change through themes via the BBWT3 menu.
  • Changes the colour scheme of; header, selected components, and menu.

Test Case

  1. User has vision problems, and struggles to view elements of web application.
  2. User navigates to themes.
  3. User can select different themes and identifies one that works best.

  4. User is able to continue using web application unencumbered.

For Testers

  1. Access the BBWT3 menu and click on "Technical Admin".
  2. Click on "Themes" from the drop down menu.
  3. Select the theme you wish to try out; the change will take place immediately.
  4. A confirmation message will be displayed in the top right corner saying "Success: Theme updated".

For Project Managers

  • Customised themes enables a user to have some preference on how they wish to use BBWT3.
  • Easily switch between themes, they take effect immediately.
  • Some colour schemes maybe easier to read for people with sight issues, such as colour blindness.
  • Themes can be overridden and new themes created for specific purposes.

For Marketing

  • Customise your look based on preference.
  • Ability to make more user friendly with options for people with vision issues such as high contrast. (Or others for people with colour blindness).
  • Custom themes can be created especially for your needs.
