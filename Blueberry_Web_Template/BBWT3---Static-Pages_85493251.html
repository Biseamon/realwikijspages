Blueberry Group : BBWT3 - Static Pages

Created by Jason Cozza on Sep 21, 2020

Overview

This feature allows for an admin to create new, edit existing, or delete individual static pages, that can be used either for the main site, or as a footer link.

  • Make new pages.
  • Edit existing pages.
  • Delete pages.
  • Use pages in the main menu.
  • Use pages for the footer.

Test Case

  1. Client needs to add in a new static page to site.
  2. Admin is able to login and navigate to "Static Pages".
  3. Admin can create new static page quickly and fill in required heading and content of new page.
  4. Admin can then navigate to Menu Customisation and add new page where needed.

For Testers

  1. Navigate to the main BBWT3 menu and click on "Operational Admin".
  2. From the drop down menu click on "Static Pages".
  3. Continue to add/edit/delete.

For Project Managers

  • Ability to quickly and simply add new webpages to the site.
  • Ability to edit and delete existing pages.
  • No extra coding or development time required.

For Marketing

  • Ability to quickly and simply add new webpages to the site.
  • Ability to edit and delete existing pages.
  • No extra coding or development time required or coding experience needed to perform these functions.
  • Changes can be made after deployment based on user feedback without additional development cost.
