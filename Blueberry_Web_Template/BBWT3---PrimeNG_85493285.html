Blueberry Group : BBWT3 - PrimeNG

Created by Jason Cozza on Sep 21, 2020

Overview

A free and open-source UI suite for Angular created by PrimeFaces. Allowing for a wide range of native widgets and components that can be integrated into BBWT3 projects.

Advantages

    • A large number of components are available.
    • Open-source and hosted on GitHub.
    • Professional premium templates are available.
    • Provides a consistent UI.
    • UI controls are fully implemented as Angular components.
    • Each control has a set of options to change appearance, behaviour, events etc.
    • Mobile-friendly and cross-browser compatible.
    • Documentation is reasonably good.

Disadvantages

    • Despite being well maintained, there are still some bugs with controls and appearance.
    • CSS selectors can be difficult to override and rules are often overriding each other, sometimes making it difficult to customise.
  • Can be quite heavy on browsers when there are lots of controls or complex logic.

FullCalendar

PrimeNG includes support for the FullCalendar library and implements it as a control.

However, the scheduler feature is a premium add-on provided by the FullCalendar developers.

Premium Templates

PrimeFaces offer multiple premium templates to be used when building applications.

Prices range from around $190-790 depending on the template used. Templates have been known to go on sale (e.g. around Black Friday.)

Ultima

BBWT3 currently uses the Ultima theme - the latest version of which can be downloaded through the PrimeFaces store (login details stored in Bitwarden as usual.)

Advantages

    • Works well with PrimeNG components as it was made by the same developers.
    • Live demo shows off some potential layouts to be used.
    • Good support for mobile devices, menu collapses on small screens.
    • SCSS source provided so we can customise and roll our own version if we need to.

Disadvantages

    • Support is only provided through a community forum (which is often quiet) but extra support is paid-for.
    • SCSS and CSS rules could do with being a little more efficient (there is a lot of overriding in the rules).
    • Can't update from a Git/npm repository, a zip file has to be downloaded and files replaced manually.

For Project Managers

  • Cut down on development time with pre-built and royalty-free components, that can be integrated into your BBWT3 web application.
  • Open-source and published under the MIT license; meaning it is available to be modified and published and sold.
  • Popular, thus having a large community of support where needed.
  • Limitation: no built-in localisation. However they provide abilities to override text through "locale" property.
  • Limitation: scheduling component of PrimeNG requires a license.

For Marketing

  • Save time and money by using existing technologies that are free to use and modify.
