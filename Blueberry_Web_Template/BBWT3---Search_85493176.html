Blueberry Group : BBWT3 - Search

Created by Jason Cozza on Sep 21, 2020

Overview

This page documents how to insert a search field within your code.


The preferred search control performs auto-completion.

Many Search controls will be found in grid filters (see below). When in a grid filter we use a different labelling technique to save space.

Example:

Example HTML:

<div class="ui-g">
    <div class="fixed-label">
        <label for="autocomplete">Label</label>
    </div>
 
    <div class="ui-g-12">
        <p-autoComplete id="autocomplete" [(ngModel)]="brand" [suggestions]="filteredBrands"
                        (completeMethod)="filterBrands($event)" placeholder="Brands..."
                        [size]="30" [minLength]="1"></p-autoComplete>
    </div>
</div>


Example Typescript:

    export class SearchComponent implements OnInit {
 
    brands: string[] = ['Audi', 'BMW', 'Fiat', 'Ford', 'Honda',
        'Jaguar', 'Mercedes', 'Mazda', 'Mustang', 'Renault', 'Volvo', 'VW'];
    filteredBrands: any[];
    brand: string;
 
    constructor() { }
 
    ngOnInit() {
 
    }
 
    filterBrands(event) {
        this.filteredBrands = [];
        for (let i = 0; i < this.brands.length; i++) {
            let brand = this.brands[i];
            if (brand.toLowerCase().indexOf(event.query.toLowerCase()) == 0) {
                this.filteredBrands.push(brand);
            }
        }
    }
}
