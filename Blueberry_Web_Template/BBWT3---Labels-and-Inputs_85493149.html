Blueberry Group : BBWT3 - Labels and Inputs

Created by Jason Cozza on Sep 21, 2020

Overview

On this page are examples of labels and inputs with code, and how it is rendered.


Label and text with a ‘hint’ text in the edit box. During editing, this hint moves/shrinks to a location above the edit text.



Example HTML:

<div class="ui-g">
    <div class="fixed-label">
        <label for="textField">Label</label>
    </div>
 
    <div class="ui-g-12 ui-md-4">
        <span class="md-inputfield">
            <input pInputText id="textField">
            <label>Edit Text</label>
        </span>
    </div>
</div>

On a page displaying read-only controls only, align the controls to the left. Try and avoid mixing editable and read-only controls on the same pane. Where this is not possible prefer showing all the controls centrally aligned.

Example HTML:

<div class="ui-g">
    <div class="fixed-label">
    <label for="text">Label</label>
    </div>
 
    <div class="ui-g-12 ui-md-4">
        <span id="text">Text Value 1</span>
    </div>
</div>


We can manage label position and colon by applying [colon] and [right] custom attributes (code available in directives folder). Both attributes are boolean attributes. Example and code below:

Example HTML:

<div class="ui-g">
    <div class="fixed-label">
        <label for="textField" [colon]="true" [right]="true">Label</label>
    </div>
 
    <div class="ui-g-12">
        <span class="md-inputfield">
            <input pInputText id="textField" />
            <label>Edit Text</label>
        </span>
    </div>
</div>
