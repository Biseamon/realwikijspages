Applying Branding to Projects 

Overview

Company Logo

The company logo and applicaiton icon can be set via the 'Customisation' tab under BBWT3 System Configuration

 

A set of logos / icons can also be set for each Company defined in the BBWT3 system. If enabled, this overrides the default set defined in System Configuration.

Colour Scheme

To apply brand colours to a BBWT3 project then you need to modify the supplied Ultima themes.

To modify an Ultima theme you need to edit the corresponding SCSS file in both of these locations:

  • project/BBWT.Client/src/assets/themes/ultima/layout/sass/
  • project/BBWT.Client/src/assets/themes/ultima/theme/

The available parameters are described below. We have also provided screenshots to illustrate the effect of each parameter.

Layout parameters

$primaryColor: #607D8B;$primaryDarkColor: #37474F;$primaryLightColor: #B0BEC5;$accentColor: #8BC34A;$accentDarkColor: #558B2F;$accentLightColor: #C5E1A5;$accentTextColor: #ffffff;$darkMenuBgColor: #424242;$darkMenuHoverColor: #676767;$darkMenuRouterLinkActiveColor: #8BC34A;$lightMenuRouterLinkActiveColor: #8BC34A;$horizontalLightMenuRouterLinkActiveColor: #8BC34A;

Theme parameters

$primaryColor: #607D8B;$primaryDarkColor: #37474F;$primaryLightColor: #B0BEC5;$accentColor: #8BC34A;$accentDarkColor: #558B2F;$accentLightColor: #C5E1A5;$accentTextColor: #ffffff;

 

Effect of template colour parameters

The effect of each template colour parameter is illustrated below. Note that each parameter can affect several different aspects of the application.

Layout File parameters

Layout File: project/BBWT.Client/src/assets/themes/ultima/layout/sass/...

$PrimaryColor

Affects page header and table footer background colours.

 

$PrimaryDarkColor

Affects page header (first block) and table header background colour.

$PrimaryLightColor

Doesn't appear in the output file... so, doesn't change anything.

$AccentColor

Affects link hover colour, menu link background colour, badge background colour (see loading time in top right).

$AccentDarkColor

Affects default link colour.

$AccentLightColor

Doesn't appear in output files... doesn't have any effect (in this theme, anyway)

$AccentTextColor

Affects highlight colour on menu buttons and badges (see top right).

$DarkMenuBgColor

Affects the background colour of menu items.

$DarkMenuHoverColor

Affects the background colour of menu items when hovering, but only inactive items.

$DarkMenuRouterLinkActiveColor

Affects active links in the menu. Doesn't seem to have any effect currently, however.

$LightMenuRouterLinkActiveColor

Same as above, no effect currently.

$HorizontalLightMenuRouterLinkActiveColor

Same as above, no effect currently.

Theme File parameters

Theme File: project/BBWT.Client/src/assets/themes/ultima/theme/...

$PrimaryColor

Affects primary colour buttons.

$PrimaryDarkColor

Affects button highlight colour.

$PrimaryLightColor

Affects PrimeNG 'multi-select' control that isn't currently used in BBWT3 anywhere.

$AccentColor

Affects paginator highlight and tab header highlight line (see arrows).

$AccentDarkColor

Affects PrimeNG file upload control, not currently used.

$AccentLightColor

Affects PrimeNG 'input switch' control, not currently used.

$AccentTextColor

Affects highlight colour of selected accordion components (amongst other components).
