Clearing SQL Data

  1. Open Remote Desktop Connection
  2. Connect to 54.171.187.136:3313
  3. Use these credentials to open a connection
    Note: Password might be changed by Sysadmins so ask them if you can’t get access
  4. Open SQL Server Management Studio 2014
  5. ShellTestMRA is Shell’s database/ShellTestMRADev is our test database
  6. Right click the dbo.Events table and Select Top 1000 Rows
  7. A query will be created add this to the bottom of the script:
    delete from [ShellTestMRA].[dbo].[Events]
  8. Press execute and the data will be deleted

Adding/Removing/Changing Attachments

  1. Create an RDP connection to the server
  2. Locate D:\IIS_ROOT\ShellTest
  3. Find the subfolder – pub-att
  4. You can remove attachments, add new ones or overwrite old ones here

Adding/Removing/Changing Images

  1. Create an RDP connection to the server
  2. Locate D:\IIS_ROOT\ShellTest
  3. Find the subfolder – pub-img
  4. You can remove images, add new ones or overwrite old ones here

Extra Notes


Shell's email security rejects BAT files. When giving an updated version of the BAT file send it to Shell as a text file.
