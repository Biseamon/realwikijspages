When I came here for work experience Martin gave some tasks to complete during the time I was here, one of which was to write a python script which cleans-up directories.

I finished it half-way through my second week here. This is the script, imporvements are always welcome.

The script can delete files based on size, extension, rename files with the same double extension, and delete files based on their last accessed date or last date of modification.

Currently it also has an automatic mode where it deletes files based on preset values, which it tells the user before-hand.
