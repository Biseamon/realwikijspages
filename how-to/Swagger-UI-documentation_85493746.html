Blueberry Group : Swagger UI documentation

Created by Jason Cozza on Nov 24, 2020

BVI's Swagger UI documentation is hosted at https://bvi-swagger.bbconsult.co.uk. Credentials are available in LastPass.

The documentation is generated at runtime through the use of swagger-ui. A Jenkins job checks out the latest version of this, alters the public/index.html to reference the BVI YAML, and then publishes it to /opt/bvi-swagger/.

To make modifications to the documentation, check out and modify the YAML from the bvi SVN project in the directory BviAws/Swagger.

Use the path bar at the top of the page to access different swagger YAML files if required.
