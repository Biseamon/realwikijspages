Blueberry Group : DB Documenter

Created by Jason Cozza on Nov 24, 2020

A tool which automatically creates wiki pages in Confluence which help to document an SQL Database.

Brief History 

Originally a Python script created by Asghar Khan and Alex Vickery, then a Confluence plugin created by Nancy Pinter. Continued by Rohan Dawson.

Download From

It's a Confluence plugin and needs to be accessed via Confluence. To document a new database, go to ‘Spaces’, ‘Space directory’, then click the little ‘i’ on the right side for the space you want to add the database to. Then go to ‘content tools’, ‘add db’.
