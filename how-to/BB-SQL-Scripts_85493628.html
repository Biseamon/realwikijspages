Blueberry Group : BB SQL Scripts

Created by Jason Cozza on Nov 24, 2020

Scripts which automatically check SQL databases and report errors or even fix them:

  • AnalyzeSlowQueryStats - analyses the data gathered by the script GetDbaStatistics and sends alerts to the user in case a query is waiting for more than 1 second.
  • CallDefragSp - defragments indexes using Ola Hallengren scripts.
  • CatchDynamicSqlParam - (created by Duncan Forsyth)  checks for a particular type of poorly written query. More information to follow.
  • GetDbaStatistics - gets the data about blocking issues and wait statistics.
  • GrantSqlPermission - grants an sql server the permission to use Amazon AWS ephemeral storage.
  • QueryChecker - (created by Duncan Forsyth) catches the majority of bad-looking queries in the development environment before they get anywhere near the live environment. More information to follow.
