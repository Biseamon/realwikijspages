Blueberry Group : BB Outlook Tools

Created by Jason Cozza on Nov 24, 2020

An add-in for Outlook 2010/2013.Provides the following features:

  • Reply with Attachments (creates a reply with the original attachments still present)
  • Tidy Contact Numbers (converts contacts' phone numbers to a unified format - now partially supported by Contact Sync, but not yet for Outlook)
  • Save Attached files (saves all incoming email attachments into a specified folder automatically when they arrive)
  • Mailing List Preview (reveals the recipient names on a mailing list by adding an extra viewing pane to messages - now supported by Contact Sync, but not yet for Outlook)

  • Signature (corrects an issue reported in the standard signature feature of MS Outlook)

  • BB Merge (allows to compare and edit several contacts - now partially supported by Contact Sync, but not yet for Outlook)

  • Contact Sync (allows to compare and merge entire contact folders - now supported by Contact Sync, but not yet for Outlook)

  • Check Duplicates (allows to check a contacts folder for duplicates and handle them - now supported by Contact Sync, but not yet for Outlook)
