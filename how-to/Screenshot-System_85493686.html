Blueberry Group : Screenshot System

Created by Jason Cozza on Nov 24, 2020

The tool allows a user to capture screenshots and create an interactive demo (a number of HTML files containing the images) as well as to export it to wiki. Shift+PrtScr is used to take screenshots. You can then change the order of the captured screens and add buttons to move to other screenshots. To be able to export to Wiki, you need to change settings in the settings.json file from the application directory,

Brief History

Created by Alex Vickery and originally tested by Neal Van Rooj who has left BB. Original version was aimed at Access Systems, and required manual linking of forms. Alex started a version to support web systems, but he hardly got that far. Martin wants to finish it. Currently Roberto Moreno is working on it for a few days. He will continue until June 20, 2016, then he'll leave BB.
