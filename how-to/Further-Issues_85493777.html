Sorting and filtering for the grids appears to be done client-side. This can lead to the client using old data if the grid is not setup to auto-refresh. There is also usually more performance overhead with JS sorting/filtering than with a database call so generally server-side sorting is considered better.MediumPerformance/Stale or incorrect data
