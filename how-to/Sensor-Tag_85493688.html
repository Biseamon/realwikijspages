Blueberry Group : Sensor Tag

Created by Jason Cozza on Nov 24, 2020

Task:

Make a reminder to keep a leg elevated using a sensor tag as well as a Raspberry PI.

Location:

https://svn.bbconsult.co.uk/repository/amp_sensor/trunk

Requirement:

  • Material: Sensor Tag and Raspberry PI
  • Raspberry Pi needs mgp123 installed.
  • Package: requirement file can be found it SVN

Instruction:

Run “is_foot_up.py” and passing in the args: The Sensor tag mac address, number of iteration to calibrate and margin at which the foot is considered to be down.

Result:

Result can be seen on this video: https://flashbackconnect.com/Movie.aspx?id=c7lrdLPrZAa5yGMeVAeELw2

Problem encounter:

The gyro values were ignored because even while not moving it reported some movement. And a calibration could not be properly made because this “noise” varies too much to be taken into account and subtracted from the final result.
