Blueberry Group : Create Tesseract Lambda Function

Created by Jason Cozza on Nov 24, 2020

Setup

Layer

The function requires certain libraries as well as the Tesseract binary.

It uses Python3.6 as runtime.

This layer can be uploaded from layer.zip.

The tesseract binary was generated by following this guide:
https://github.com/gwittchen/lambda-ocr

Function

The code for the function can be found in the function folder.

It uses Python3.6 as runtime.

Function Role

The function needs the following permissions:

  • AmazonS3FullAccess

  • AWSLambdaBasicExecutionRole


Settings

The function might timeout if the basic settings are left unchanged. It will need more memory as well.

Result

The function generates a text file out of the image in the same bucket and same directory.

Problem

The function uses a copy of Tesseract TrainedData. This is not ideal and should be changed.

The function must be in the same region as the S3 Bucket in order to add triggers.
