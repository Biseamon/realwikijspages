Blueberry Group : Blueberry Outloot

Created by Jason Cozza on Nov 24, 2020

Blueberry Outloot is an Outlook plugin designed to help Blueberry's communication being more efficient.

It is a simple extension developed in C# including these different features : 

  • PTS Helper to help you logging any issue in the PTS based on a customer email.
  • Newsletter Template to create quickly your own newsletter. An easy way to send updates to the customers about their ongoing projects with us.
  • Watchdog to help you answering your emails in a certain delay.

If you want to know more about these different features, please have a look to the related articles.


How to install :

The installation of this add-in is very simple. To know more about this, please have a look at the related article : Outloot Installation Steps.
