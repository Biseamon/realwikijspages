Blueberry Group : Student / trial / background projects

Created by Jason Cozza on Nov 24, 2020

A summary of projects developed by students and trialists as well as the background projects.



  • AutoDeploy   
Tool to deploy BBWT web sites to servers, integrated with PTS. Mostly finished, waiting testing - probably implemented last by Alex Vickery. May be superceeded by BBWT3
  • BB Outlook Hours Plugin
Tool to show BB people the availability and working hours of other BB people. Also shows members of mailing lists. Very close to completion, needs more testing and release
  • BB Outlook Tools
Old tool which supported multiple useful features for OL users and BB people. Written originally by Kevin Lazenby.
  • BB SQL Scripts
SQL Performance check scripts written by Jairo. Very useful. Deployed but not fully documented or published.
  • BB Support Plugin (Blueberry Outloot)
Plugin created by student Victor Simiart for BB Systems to use for sending PTS mails. Not sure of status.
  • Contact Sync
Tool to compare synchronise two google contacts lists. Worked, not finished.
  • Database Maintainer
Old simple tool to update DB when releasing to live. Kevin Lazenby.
  • DB Documenter
Confluence plugin created by Nancy Pinter to automatically document SQL databases in Confluence. Finished, works - but not as useful as hoped.
  • Google Sync Checker
Tool which compares local sync of Google docs with Server via API, and finds discrepancies. Worked, needed a bit more work - could be released.
  • LogSys
Old BB tool for client-side software, e.g. FB to handle logging. Kevin Lazenby
  • Screenshot System
Complex tool to capture screenshots from applications and create confluence pages from them, including active links between the screens. Worked for Access Applications, but not for web applications. Needed more work.
  • SimpleTV
An effort to make a TV that a grandpa could use.
  • Telephone Number Checker
