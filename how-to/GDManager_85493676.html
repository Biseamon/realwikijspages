Blueberry Group : GDManager

Created by Jason Cozza on Nov 24, 2020

The google drive manager is the core object which contains all of the file and folder information.

Instruction:

Once the environment is set up call the connect function in function.py which will return the service (the client_secret.json need to be in the same folder as function.py).

Then create a new GDManager by passing the service object and the email of the owner of the drive.

On creation, the GDManager will get all files and folder information from google drive and then start drawing their paths.

Once GDManager has finish initialising you can call its sort method to have it sort all the files and folder in different dictionaries.

The Dictionaries are written as so: {“file/folder ID”: file/folder Object}
