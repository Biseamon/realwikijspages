Blueberry Group : Google Sync Checker

Created by Jason Cozza on Nov 24, 2020

The application checks if files on the local Google drive folder were correctly synchronised with their cloud copies by Google docs sync.

Brief History

Developed by John Nathan, last change probably in 2014. Continued by Vincent Goodman in June 2016.

Technology used

C#, .Net using Google API

SVN Repository

https://svn.bbconsult.co.uk/repository/drivetools 

PTS Project Name/Code

DriveTools

Current Status

The project has been mostly finished in 2014 although some issues left untouched. Based on the data found in PTS, no one has been working on the project after it. The application was tested by Martin Green only. Currently being tested by Julia McKenzie.
