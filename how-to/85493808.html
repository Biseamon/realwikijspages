Blueberry Group : Prepare a new employee's laptop

Created by Jason Cozza on Nov 24, 2020

This tuto is designed to help the setup of a new employee's laptop

Windows Installation 

If the laptop is new , you can boot into the usb stick with the blueberry's iso image on it

when the installation process is ready , you will be able to create a new user, you can use his first name and his last name

For the moment, you don't need to create a password, the user will setup one later


Windows configuration


Updates

You can now configure the wifi on this laptop and check for any updates, install every updates on this devices.

Office tools (Word, Excel, Powerpoint)

  • Exit all your Office applications such as Outlook, Word, Excel, Powerpoint.
  • Download o2019.zip ( https://drive.google.com/file/d/1KvDdFwFh7akKQg3TT5-7bIYHiPxjgmSg/view?usp=sharing) (2GB download, 3GB when uncompressed, you need 10GB free space)
  • From the Start menu, type File and open File Explorer
  • Navigate to your Downloads folder. Confirm that you can see o2019.zip
  • Double-click o2019.zip . Use the provided password. You should see the o2019 folder inside it.
  • Select the o2019 folder and copy it by holding down the CTRL key and pressing C
  • Select the Back or Up icons so that you return to the Downloads folder
  • Paste the o2019 folder by holding down the CTRL key and pressing V
  • The o2019 folder should now be inside the Downloads folder
  • Start a command prompt (black window) by pressing the Windows key (start menu) and typing CMD then press the Return key.
  • Type:
cd Downloads
cd o2019


1. If you just want Outlook, Word, Excel and Powerpoint:

setup.exe /configure


Press RETURN at the end of the line. This should begin installation 

immediately, with no need to wait to download anything else.

2. If you want Access as well as Outlook, Word, Excel and Powerpoint:

setup.exe /download configuration-access.xml
setup.exe /configure configuration-access.xml


Press RETURN at the end of each line. There will be a long wait 

after the /download command, while it downloads Access. Nothing
will appear on the screen during this download. You can use Task
Manager to monitor network and storage activity.

3. If you want all the Office applications:

setup.exe /download configuration-all.xml
setup.exe /configure configuration-all.xml


When asked "Do you want to allow this application to make changes
to your device?" click YES.

This should then install Office 2019 for you.

If it complains that it cannot install 64-bit applications because
you have 32-bit applications installed, please uninstall your
32-bit Office applications manually, then rebooting. If this fails,
you can use the included SetupProd_OffScrub.exe program to
completely uninstall all Office applications, but be warned that
this may take an hour or more, and the program does NOT display any
progress (or indeed anything at all) for tens of minutes at a time
(you can view activity in Task Manager). We recommend trying to
manually uninstall 32-bit Office applications, and rebooting,
before using the SetupProd_OffScrub.exe tool as a last resort.

Once installed, Office will activate automatically with Blueberry's
Pro Plus 2019 Volume licence.
