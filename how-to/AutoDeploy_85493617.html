Blueberry Group : AutoDeploy

Created by Jason Cozza on Nov 24, 2020

A Windows application connected to PTS and Amazon S3 and can automatically install a new website version on a server. Once it detects a new web site version committed to PTS (and added to Amazon), the service receives a notification that a new version was released, downloads it from Amazon and installs/updates the current machine. If necessary, it also updates the database.
