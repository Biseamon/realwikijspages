Blueberry Group : Outloot Installation Steps

Created by Jason Cozza on Nov 24, 2020

First, download the installer by clicking the following link : <NoLinkYet>

Your folder should contain this :


Blueberry Outloot installation folder.

Please, be sure to close Outlook before installing the add-in. You can uninstall any precedent version if you want (Control Panel > Programs and Features > BlueberryOutloot) but new versions should erase any former one.

Launch the setup by double-clicking on it.
The installer may need an internet access to validate the certificate so you may be issuing problems if you are not connected to the Internet.


This is the normal certificate validation message that you should have.

Just click the Install button to install the add-in.

The add-in should be running as soon as you will start Outlook. You can check by looking for the new tab called "BLUEBERRY" on Outlook's homepage.

The new tab created by the add-in.


NB: If you think the add-in may not be running, here is how to check its current state :

          Go to File > Options > Customize Ribbon and tick "Developer" in the right list.
          Click the new tab "DEVELOPER" that appeared on your homepage.
          Click "COM Add-Ins"
          Check that "BlueberryOutloot" is ticked in the list.
