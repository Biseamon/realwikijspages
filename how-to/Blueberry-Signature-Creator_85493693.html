Blueberry Group : Blueberry Signature Creator

Created by Jason Cozza on Nov 24, 2020

This is a tool for generating appropriate signatures for use in MS Outlook.


The signature tool has the following features:


  • Allows signatures to be created quickly and easily.
  • Template for use with BB Consult, BB Systems and BB Software.
  • Creates both a standard and 'Reply' signature which can be edited as required in Outlook.

Once downloaded and installed, you’ll need to take the following steps:

 

 

1)     Choose which Blueberry Company you wish to create signatures for by ticking the checkboxes:

 

 

 

2)     Select the template that you wish to use using the tabs (BB Consult, BB Software or BB Systems). Your template preview will appear below them:

 

 

 

 

 

 

3)     Fill in the fields and click ‘Create and Save’ once you’ve finished:

 

 

Note – if you wish to use different template tabs for each of the Consult/Software/Systems options, you’ll need to use the ‘Create and Save’ button each time after selecting a template tab.
