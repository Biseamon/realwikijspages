Create Lamdba Function


Before deploying Lamdba Backup functions you should create it first by following instructions below:


  1. Open Amazon web console and then click on services, then click on Lamdba
  2. Click on Create Function and then click on "batch-get-job-python27", then you are going to see "Configure triggers" GUI, then click Next
  3. Fill the information below, like it is shown on the screenshots below:



    Make sure to use the name of the lambda function here before the ".lambda_handler" text


    On Advanced settings Tab make sure to use 5 min, at the time of this writting it is  maximum execution time



Deploy Python Backup scripts


To deploy files ConsumerCoreBBBackupScript.py and ProducerCoreBBBackupScript.py follow the instructions below:


  1. Log in on the ubuntu server called test-ansible

  2. Execute the command "sudo -i"

  3. Execute the command "cd lambda"

  4. Update the code of the files ConsumerCoreBBBackupScript.py and ProducerCoreBBBackupScript.py
  5. To deploy the file called ProducerCoreBBBackupScript.py execute the script called deployProducerCoreBBBackupScript.sh
  6. To deploy the file called ConsumerCoreBBBackupScript.py execute the script called deployConsumerCoreBBBackupScript.sh
