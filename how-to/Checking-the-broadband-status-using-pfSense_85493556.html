Blueberry Group : Checking the broadband status using pfSense

Created by Jason Cozza on Nov 24, 2020


If you notice slowness in the internet speed you're getting, there could be packet loss or other issues with the main CoreSpeed line.


You can check CoreSpeed's status at https://status.corespeed.cloud.

You can check Andrews and Arnold's status at http://aastatus.net/.



Corespeed has 60Mbps down, 30Mbps up.

Andrews and Arnold's has 8Mbps down, 1Mbps up.


Step-by-step guide

  1. Navigate to https://10.16.1.1:8080 or https://mt-hq.bbconsult.co.uk:8080. Accept any certificate warnings.
  2. Use the BitWarden note 'pfSense Consult HQ (office-user)'.
  3. Observe the dashboard, in particular the interface and gateway statuses. WAN_PPPOE is the main CoreSpeed line, while WANAA_PPPOE is the backup Andrews and Arnold line.
  4. When the main line is down or experiencing severe packet loss, pfSense will route traffic through the backup line.
  5. If the main line is down, give CoreSpeed a call and give them an estimate of when the outage occurred.
  6. If the main line is experiencing a lot of packet loss, it might be worth calling them to find out what's going on.

 


You can contact CoreSpeed either at support@corespeed.cloud or by calling ‭0330 004 1000‬.

You can contact Andrews and Arnold support at either support@aa.net.uk or by calling 033 33 400 999.
