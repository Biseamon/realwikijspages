Blueberry Group : Connecting to HQ VPN Server

Created by Jason Cozza on Nov 24, 2020

These instructions describe how to connect to HQ VPN server to get direct access to HQ LAN.

Requirements: OpenVPN v2.0 or higher.

  1. Setup OpenVPN client on your PC.

  2. Go to http://openvpn.net/index.php/open-source/downloads.html and download appropriate version of OpenVPN. For most users it should be Windows Installer (64-bit). When the download is complete, run the installer and install OpenVPN using default options.

  3. Download hq.bbconsult.co.uk.ovpn from here and copy to C:\Program Files\OpenVPN\config\. config dir must exist inside the folder which you specified when installed OpenVPN.

  4. Press Start -> Programs -> OpenVPN -> OpenVPN GUI. In the right bottom corner of the screen right click on OpenVPN GIU icon and chose Connect. If you get error "Error creating HKLM\SOFTWARE\OpenVPN-GUI key", right click om OpenVPN GUI icon and chose "Run as Administrator".

  5. When prompted, enter login and password given to you by sysadmins


Note for sysadmins: OpenVPN Windows network adapter can have /30 mask only. That's why you need specify local and remote address for each VPN user inside /30 subnet.
