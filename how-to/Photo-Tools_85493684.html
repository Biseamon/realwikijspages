Blueberry Group : Photo Tools

Created by Jason Cozza on Nov 24, 2020

Task:

Copy photos to a sorted folder by date. Ex: 2016\2016-12-31\...

Location:

https://svn.bbconsult.co.uk/repository/photo_tools/trunk

Requirement:

Environment: Python 3.4

Packages: requirement file can be found on SVN.

Instruction:

Edit the Paths in Config.ini file to match the proper incoming, log, result, and error folder.

Run PhotoSorting.py

Another option would be to run RunScript.py which will then check for new files in the incoming folder but it is running in a console so will stop once the computer is turned off and wont restart on start up like a service.

Result:

Photo are move to either the result folder or error folder if an error occurred or if the photo already exists.
